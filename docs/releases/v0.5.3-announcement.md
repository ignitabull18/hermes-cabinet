# Cabinet v0.5.3 announcement

Publish this copy only after the GitHub Release, npm packages, native artifacts,
manifest, and public download URLs have passed the release gates.

## Discord

🗄️ **Cabinet v0.5.3 is live**

This release makes setup and desktop delivery much smoother:

- Guided AI-provider setup inside Cabinet
- A focused three-step Slack setup guide with screenshots and connection testing
- New Google Ads, Meta Ads, and StackAdapt integrations
- Drag-and-drop files and folders with progress and uploads up to 1 GB
- Native macOS DMG/ZIP and Windows installer/ZIP downloads

The macOS build is signed and notarized. The Windows build is currently unsigned,
so Windows may show an Unknown Publisher or SmartScreen warning; it is still
installed, launched, health-checked, and uninstalled automatically on a native
Windows runner before publication.

Download: https://runcabinet.com/#get-started

Release notes: https://github.com/cabinetai/cabinet/releases/tag/v0.5.3
